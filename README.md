# European Monarchies Timeline

Project to collect, model and visualise European monarchies from the Roman era through 2026.

How to use

- Read SPECS.md for data schema and scope.
- Add or update data in data/*.json or data/*.csv.
- Run the data cleaning scripts (scripts/) to build timeline data/.
- Open docs/index.html (or host on GitHub Pages) for the visualisation.
